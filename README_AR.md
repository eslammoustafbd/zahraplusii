# Zahrat El-Rabea For Food Manufacturing

نسخة إنتاج نهائية لموقع متعدد اللغات ولوحة إدارة على المسار الداخلي `/eslammm`، مبنية بخادم Node.js دون حزم تشغيل خارجية.

## خصائص الإصدار

- واجهة رسمية بخط Noto Kufi Arabic للعربية وManrope للغات اللاتينية، مع بدائل نظامية عند تعذر تحميل الخط الخارجي.
- Landing ديناميكية تدعم الصور والفيديو والـPoster والترتيب والتفعيل من لوحة الإدارة.
- إدارة الشعار، الهوية، بيانات التواصل، الفوتر، الخريطة، المنتجات، التصنيفات، الشهادات، الإنجازات، الأخبار، الأسئلة الشائعة وSEO.
- دعم العربية والإنجليزية والإسبانية والإيطالية.
- تصميم مرن لجميع المقاسات، وتحسينات للأداء والثبات البصري والوصول بلوحة المفاتيح.
- ضغط Brotli/Gzip، ETag، دعم Range للفيديو وPDF، رؤوس أمان وملفات Sitemap وRobots ديناميكية.
- إعدادات نشر عامة لـNode.js وDocker وPM2 وProcfile وNginx وApache Reverse Proxy.
- واجهة الموقع العامة لديها محتوى احتياطي ثابت، بينما تحتاج وظائف لوحة الإدارة والنماذج إلى تشغيل خادم التطبيق.

## متغيرات البيئة

`HOST` و`PORT` و`SITE_URL`، مع نموذج جاهز داخل `.env.example`.

## التحقق

الأمر `npm run check` يفحص JavaScript وJSON والأيقونات والأصول والمسارات الأساسية قبل النشر.
